# 🎬 MoviesHub

A modern Movie Database web application built using **HTML, CSS, and JavaScript**, powered by **The Movie Database (TMDB) API**. Browse trending movies, search instantly, explore by genre, watch trailers, and save your favorite movies.

## 🚀 Features

- 🔍 Live Movie Search
- 🎬 Browse Popular Movies
- 🎭 Filter Movies by Genre
- ⭐ Movie Ratings
- 📅 Release Date
- ❤️ Add & Remove Favorites
- 💾 Favorites saved using Local Storage
- 🏠 Home Button
- 📄 Pagination
- 🎥 Watch Official Trailer
- 🌙 Dark Theme UI
- 📱 Fully Responsive Design
- ⚡ Smooth Hover Animations
- 🎯 Modern Netflix-inspired Interface


Go to the project folder

Open `index.html` in your browser.

---

## 🔑 API

This project uses **The Movie Database (TMDB) API**.

Get your free API key from:

https://www.themoviedb.org/settings/api

Replace your API key inside `script.js`.
